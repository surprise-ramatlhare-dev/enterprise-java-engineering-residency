# Day 1 Reflection

## What did I learn today?

Today marked the official beginning of my 120-Day Enterprise Java Engineering Residency. I successfully set up my professional development environment by configuring Git, GitHub, SSH authentication, and creating my engineering repository.

On the Java side, I learned the fundamentals of the Java Collections Framework. I understood the differences between arrays and collections, why collections are preferred in enterprise applications, and when to use the different collection types. I learned how to use List, ArrayList, LinkedList, Set, HashSet, TreeSet, Map, HashMap, and TreeMap through practical coding exercises.

I also learned the importance of programming to an interface (for example, using `List` instead of `ArrayList`) and completed multiple Java programs to reinforce these concepts.

---

## What was the most challenging topic?

The most challenging concept today was understanding the differences between the various collection implementations, especially knowing when to choose ArrayList, LinkedList, HashSet, TreeSet, HashMap, and TreeMap. Understanding that `Map` is not part of the `Collection` interface and learning how the different data structures work internally required careful attention.
